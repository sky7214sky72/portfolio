# 영단어 외우기 사이트

예전에 만든 영단어 만들기 사이트 TDD 연습 겸 새롭게 만들어보려고 합니다

## 개요

 - 관리자 기능 (영단어 등록, 엑셀 업로드를 이용한 영단어 일괄등록, 영단어 삭제, 영단어 수정)
 - 사용자 기능 (영단어 조회, 외운 영단어 저장, 외운 영단어 삭제, 외운 영단어 통계 조회)
 - 공통 기능 (로그인-카카오, 네이버, 구글, 로그아웃)

## 프로젝트 사용 기술 스택

Spring boot, JPA, postgresql, H2 사용 예정
